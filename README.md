## Prerequisite

- Git
- Node.js (v22.16.0 and above)
- npm

## Clone project using CMD (Windows) or Terminal (Linux/MacOS)

```bash
git clone https://github.com/thampaponn/nestjs-encryption-api.git

cd nestjs-encryption-api
```

## Project setup

```bash
$ npm install
```

## Start service

```bash
$ npm run start
```

## Run unit tests

```bash
$ npm run test
```
